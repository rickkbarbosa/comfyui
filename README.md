COMFY-UI
=======

[ComfyUI](https://www.comfy.org/) is a graphical node-based interface for creating images, videos and audio. It requires minimal coding and lets the user create diffusion workflows by dragging and dropping nodes in a visual interface. It is open source, and can be installed on Windows, Mac OS and Linux.


This repository launches COMFY to use with ROCM AMD GPU
